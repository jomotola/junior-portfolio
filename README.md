# Omotola's Personal Website 💖

A personal portfolio website showcasing my projects and skills as an aspiring software engineer. It uses a soft pink aesthetic and focuses on clean design, responsiveness, and user-friendly interactions.

## Features ✨

- Responsive design for desktop and mobile
- Sticky glassy navigation bar
- Project showcase with live links
- Contact form powered by Formspree
- Custom cursor and aesthetic UI elements

## Built With 🛠️

- HTML5
- CSS3
- JavaScript
- Formspree (for contact form)
